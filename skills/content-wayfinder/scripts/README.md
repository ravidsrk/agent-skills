# Scripts for `content-wayfinder`

Local copies so worktrees can run helpers without monorepo-root path issues.

- `spawn_worker.sh` — Orca terminal + `dispatch --inject` + claude Enter fix
- `preflight.py` — BASE ≠ default branch, git/gh present
- `pm.py` — tolerant inbox/check JSON helper

**These call Orca — they do not replace it.** Load the Orca `orchestration` skill for command grammar.
