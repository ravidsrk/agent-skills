# Matt coding flow (v1.1+)

Authoritative for software work:

```
/wayfinder → /to-spec → /to-tickets → /implement
```

- Wayfinder charts **decisions** until the way is clear.
- Then freeze a **spec** and ticket it.
- Hand **implementation to AFK agents** (Orca fleet) so the human can context-switch.

Wayfinder as the *entire* delivery path is fine for **non-coding** (see `content-wayfinder`).
For coding, prefer exiting wayfinder at map-complete into to-spec.
